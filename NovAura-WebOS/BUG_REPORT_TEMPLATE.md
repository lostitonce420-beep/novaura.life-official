# 🐛 NovAura Bug Report Template

For everyone posting "it broke" - here's how to get it fixed FAST:

## Quick Report (Copy/Paste)

```
**What I was trying to do:**
[One sentence]

**What happened:**
[Error message or behavior]

**Browser:**
[Chrome/Firefox/Safari/Edge]

**Screenshot:**
[Drag image here]
```

## Priority Tiers

### 🔥 CRITICAL (Fix Today)
- Complete crash/freezes
- Data loss
- Security issues
- Can't login at all

### ⚠️ MAJOR (Fix This Week)  
- Feature completely broken
- Major UI glitches
- Performance issues

### 💅 MINOR (Fix Eventually)
- Visual inconsistencies
- Minor UX friction
- Typos
