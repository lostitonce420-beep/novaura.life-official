Welcome to your AI-powered desktop operating system! This guide will get you from zero to building in minutes.

## 🎯 First 5 Minutes

### 1. **Take the Tour** (30 seconds)
When you first open Novaura, you'll see an interactive onboarding. Don't skip it! It shows you:
- Where your apps live
- How to open your first project
- How to use Nova AI

**Pro tip:** You can restart the tour anytime from Settings > Help.

We personalize your experience based on your goals:

| If you want to... | Choose | You'll get |
|-------------------|--------|------------|
| Learn coding | **Learn to Code** | Beginner tutorials, guided projects |
| Build apps/games | **Build Projects** | Templates, full IDE access |
| Ship production code | **Professional** | Advanced tools, custom domains |

### 3. **Open Your First App** (1 minute)
Click any icon in the sidebar:
- 💻 **Code IDE** - Start coding immediately
- 🎨 **Vertex AI** - Generate images/videos  
- 💬 **AI Chat** - Ask Nova anything
- 🌐 **Browser** - Surf the web

**Try this:** Open the Code IDE and type:
```javascript
console.log("Hello Novaura!");
```

### 4. **Chat with Nova** (2 minutes)
Nova is your AI assistant. She's always available in the chat bar at the bottom.

**Try these prompts:**
- `"Create a simple calculator app"`
- `"Generate an image of a futuristic city"`
- `"Explain what React hooks are"`
- `"Deploy my project to novaura.life"`

### 5. **Create Your First Project** (30 seconds)
- Click **+ New Project** on the welcome screen
- Choose a template (or start blank)
- Give it a name
- Start building!

## 📚 Core Concepts

### The Desktop
Think of Novaura like Windows or macOS, but in your browser:
- **Sidebar** - Your apps (customize which ones appear)
- **Windows** - Each app opens in its own draggable window
- **Taskbar** - Quick access to frequently used tools
- **Notifications** - Top-right alerts and updates

### Apps & Windows
Unlike regular websites, Novaura apps are **persistent**:
- Open an IDE → Write code → Switch to browser → Come back, code is still there
- Minimize windows to taskbar
- Drag windows around, resize them
- Work on multiple things at once

### Nova AI
Your personal assistant that:
- **Writes code** for you
- **Generates images/videos** 
- **Explains concepts** in simple terms
- **Debugs errors** you encounter
- **Deploys projects** with one command

**Pro tip:** The more you use Nova, the better she understands your style (if you enable memory).

## 🌟 Power User Tips

### Keyboard Shortcuts
| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Open Command Palette |
| `Cmd/Ctrl + N` | New Project |
| `Cmd/Ctrl + O` | Open File |
| `Cmd/Ctrl + S` | Save |
| `Cmd/Ctrl + Shift + P` | Deploy Project |
| `Cmd/Ctrl + /` | Toggle AI Chat |
| `Cmd/Ctrl + B` | Toggle Sidebar |
| `Esc` | Close Active Window |

### Command Palette (`Cmd/Ctrl + K`)
Your superpower! Search and execute anything:
- Apps: "IDE", "Browser", "Vertex"
- Actions: "New Project", "Deploy", "Settings"
- Files: Search your projects
- Commands: Anything!

### Personalize Your Desktop
1. Open **Settings** (⚙️ icon)
2. Go to **Personalization** tab
3. Choose your theme:
   - **Cosmic** - Deep space (default)
   - **Blue Night** - Navy sunset vibes
   - **Aurora** - Light mode
4. Customize your taskbar (pick which apps show)
5. Rearrange apps by dragging

### Enable AI Memory (Optional)
Want Nova to remember your preferences?
1. Go to **Settings** > **AI** > **Memory**
2. Toggle **"Enable Semantic Engrams"**
3. Nova will learn your style over time

**Privacy note:** Memory is stored locally on your device, not in the cloud.

## 🚀 Deploying Your Project

### One-Click Deploy
1. Open your project
2. Click **Deploy** button (or type in chat: `"Deploy this"`)
3. Choose a domain:
   - `yourname.novaura.life` (free)
   - Connect your own domain (Pro)
4. Share your live site!

### What Gets Deployed?
- Web apps (React, Vue, HTML)
- Static sites
- APIs (Node.js, Python)
- Games

**Not supported yet:** Databases (use external), long-running servers.

## 🚦 Troubleshooting

### "Something isn't working"
1. Refresh the page (F5)
2. Check your internet connection
3. Clear browser cache (Ctrl+Shift+R)
4. Try a different browser
5. Contact support

### "AI isn't responding"
- Check if you've hit your daily limit (Free tier)
- Try a simpler prompt
- Wait a moment and retry
- Check Nova's status: status.novaura.life

### "My project won't deploy"
- Make sure it has an `index.html` or valid entry point
- Check for syntax errors
- Try deploying again
- Contact support with error message

### "Windows are laggy"
- Close unused windows
- Disable particle effects in Settings > Performance
- Use Chrome/Edge for best performance
- Check your computer's RAM usage
